# Chatbot Project 🤖

## 📌 About the Project

This is a simple chatbot application built using React. It responds to user messages based on predefined patterns. The goal of this project is to understand how chat interfaces work and how we can process user input to generate responses.

Currently, the chatbot can handle basic queries like greetings, current date, and simple fun commands. In future, I plan to improve it by adding AI-based responses.

---

## 🚀 Features

* Responds to basic greetings like **Hi, Hello**
* Can tell the **current date**
* Fun feature like **flip a coin**
* Real-time response system
* Clean and simple UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* React

## ⚙️ How It Works

* User types a message in the chat input
* The app checks the message against predefined patterns
* Based on the match, it returns a response instantly

## 💻 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/your-username/chatbot-project.git
```

2. Go to the project folder

```bash
cd chatbot-project
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```

5. Open in browser

```
http://localhost:5273
```
